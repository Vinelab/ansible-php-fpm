# Ansible PHP-FPM
Install PHP-FPM on Centos/Red Hat with Ansible

## Installation
Clone this repository inside your ```roles``` direcotry

## Synopsis
will install the listed services from [remi repo](http://rpms.famillecollet.com):

```yaml
- php
- php-fpm
- php-mbstring
- php-mcrypt
- php-pdo
- php-mysql
- php-xml
- php-gd
```
